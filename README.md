# Cocktail Recipe Finder 🍹

This is a simple web application that allows users to search for cocktail recipes using an API.

## Features
- Search cocktails by name
- Fetch data from API
- Display results dynamically

## Technologies Used
- HTML
- CSS
- JavaScript

## How it works
User enters a drink name → app fetches data from API → displays results dynamically.

## Future Improvements
- Improve UI design
- Add filters
- Save favorite drinks
